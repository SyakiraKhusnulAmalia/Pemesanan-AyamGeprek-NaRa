# Ayam Geprek NaRa

Website pemesanan online untuk usaha kuliner Ayam Geprek NaRa, dibangun 
sebagai proyek UTS mata kuliah Pemrograman Website E.

## Live Demo
https://visionary-meringue-013694.netlify.app/

## Fitur

- **Home** — Halaman utama dengan tampilan menu favorit dan tombol pesan
- **Tentang** — Informasi usaha, keunggulan produk, dan detail operasional
- **Menu** — Daftar menu lengkap (Ayam, Camilan, Minuman) dengan fitur tambah pesanan
- **Promo** — Penawaran spesial dan paket hemat
- **Pesan** — Form pemesanan dengan detail pesanan dan total harga
- **Riwayat Pesan** — Daftar pesanan diproses dan pesanan selesai

## Teknologi

- HTML
- CSS
- JavaScript

## Cara Menjalankan

1. Clone repository ini
2. Buka file `index.html` di browser
3. Atau akses langsung via link live demo di atas
